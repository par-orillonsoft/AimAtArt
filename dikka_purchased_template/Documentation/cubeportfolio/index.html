About

Cube Portfolio is a powerful WordPress plugin that provides beautiful animated filtering, custom captions and it's perfect for portfolios, galleries, team members, blog posts or any other ordered content.

Browser support:

  • Chrome
  • Firefox 4+
  • Internet Explorer 8+
  • Opera
  • Safari

You can get started by watch the introductory video.

How to install

At the root folder of the downloaded zip from CodeCanyon you will find a zip file (cubeportfolio.zip) that can be installed on your Wordpress site. This file contains the plugin data. After you've located this file you can start the installation of the plugin.


1. First, go to Plugins page and click the Add New button.

2. Click the Upload link.

3. Click the Choose File an then select the cubeportfolio.zip file.

4. Finally, activate the plugin by clicking on the Activate Plugin link.

Create a portfolio

To create a new portfolio go to Cube Portfolio menu from the admin page and then click on the blue button Add new portfolio

From the popup choose a template that is best for you and then use the live template builder to customize the look and feel.

Also, from the plugin's main page you can duplicare, remove or activate a plugin.

To edit a portfolio you can click on the plugin's name and the live template builder will show for you.

After you are done editing the plugin, don't forget to click the Save template button to save your changes.

Add a portfolio to your site

You can add portfolios to your pages/posts via the shortcode method: [cubeportfolio id="1"]. The id paramenter represent the unique id (required) for a portfolio.

You can insert any number of portfolios on your pages/posts using this shortcode method.

Template & style

The plugin comes with 6 templates for witch you can extend them to look and feel as you wish.

After you select a template you can live customize that template with a lots of options.

When you change the style for a template, only that template will look as you choose. The style for default templates will remain the same. To extend from a template already customize, go to Cube Portoflio admin main page and duplicate that portoflio.

Cube Posts - Custom Post Type

To create singlePage or singlePageInline popup you must create new entries for `Cube Posts` type. To link an item of the portfplio to the `Cube Posts` type watch this video.

Global Settings

The global settings are specific to the plugin itself. You can preload CSS and Javascript files to be available only for the pages that use Cube Portoflio plugin.

How to update

At the root folder of the downloaded zip from CodeCanyon you will find a folder named cubeportfolio. This folder contains the plugin data.

With your preferred FTP client software (FileZilla, Cyberduck, etc) connect to the remote server where your WordPress website is hosted.

After you are connected to your remote server, navigate down to the /wp-content/plugin/ directory of WordPress website, where you can find cubeportfolio plugin installed.

Select the cubeportfolio plugin directory and choose delete option to remove the outdated version. You will not loose plugin settings/data.

Then, select the folder named cubeportfolio downloaded from codecanyon that is still on your PC and put it on /wp-content/plugin/ directory.

Cube Portfolio is now up-to-date and should be working correctly.

Import/Export

Import/Export feature is useful when you want to transfer a grid from one wordpress site to another. The export will generate a .json file with Cube Portfolio grids and settings from database. At import you must provide a .json file from a previous export.

Note When you import a file it's always best to make sure that you have the same version number on both sites. Please update the plugin on both sites before doing an export -> import.
